### 📋 **快速開始步驟**

#### 👨‍🏫 **老師部分（初次設置）**
1. **進入管理員設定** - 點擊「管理員設定」
2. **輸入管理員密碼** - `[管理員密碼]`
3. **設置 OpenAI API KEY** - 從 https://platform.openai.com/api-keys 獲取
4. **保存設定** - 點「保存設定」
5. **選擇分享方式**（二選一）：
   - 🔗 **方案 A**：點「🔗 分享連結」→ 複製長連結
   - 🔐 **方案 B**：點「🔐 設置碼」→ 複製短碼
6. **分享給全班** - 郵件、QR CODE、班級群組等

#### 👨‍🎓 **學生部分（每次遊玩）**

**使用分享連結（方案 A）：**
1. 點擊老師分享的連結
2. 輸入暱稱和遊戲密碼（`game2026`）
3. 直接開始冒險 🎮

**使用設置碼（方案 B）：**
1. 進入遊戲主頁
2. 點「或輸入老師的設置碼？」
3. 輸入老師提供的設置碼（如 `SETUP-ABC1234`）
4. 輸入暱稱和遊戲密碼（`game2026`）
5. 開始冒險 🎮

#### 遊戲過程
1. **與 11 位導師互動** - 每位導師有多個選擇
2. **完成遊戲** - 生成屬性雷達圖、升學建議、AI 圖像提示詞
3. **複製結果** - 將提示詞貼到繪圖工具生成圖像

## ✨ 核心特色

### 🎮 沉浸式 RPG 體驗
- **11 位導師互動**：企業家、設計師、教育工作者、科學家、程式設計師、醫生、金融家、電影製片人、律師、建築師、環保工作者
- **50+ 故事選擇點**：動態分支敘事，每次遊玩體驗都不同
- **多條故事路線**：根據選擇走出獨特的職業探索之旅

### 📊 屬性系統 & 雷達圖
- **6 維屬性追蹤**：創意力、分析力、領導力、同理心、勇氣、好奇心
- **六邊形雷達圖**：視覺化展示能力分布
- **實時屬性累積**：每個選擇都影響最終屬性分數

### 🤖 AI 驅動的結果生成
- **個性化升學描繪**：基於 250+ 字的深度分析
- **AI 圖像提示詞**：可用於 DALL-E、Midjourney 等 AI 繪圖工具
- **科系推薦**：針對屬性和職業傾向的具體建議

### 🎵 內置背景音樂系統
- **本地文件上傳**：支持 MP3、WAV 格式
- **免費音樂來源指南**：內置三個免費音樂網站快捷連結
  - 🎨 Pexels Music
  - 📚 Free Music Archive
  - 🎬 YouTube 音樂庫
- **完整音樂控制**：播放、暫停、音量調整

### 📱 響應式設計
- 完美適配手機、平板、電腦
- 浮動音樂控制面板
- 清晰的視覺層級和使用者體驗

## 🎮 遊戲流程

```
登錄 (輸入暱稱 + 密碼 game2026)
  ↓
故事介紹與背景設定
  ↓
遇見企業家李成功 (3個選擇) → 2條分支
  ↓
遇見設計師林創意 (3個選擇) → 2條分支
  ↓
遇見教育工作者陳愛心 (3個選擇)
  ↓
自由選擇探索其他 8 位導師（可選）
  ↓
AI 分析所有選擇和屬性
  ↓
生成結果：
  ├─ 六邊形屬性雷達圖
  ├─ 個性化升學描繪
  ├─ AI 圖像提示詞
  └─ 科系推薦 + 行動建議
```

## ⏱️ 遊玩時間

| 標準路線（5-6個導師） | 15-20 分鐘 |
| 完整冒險（所有導師） | 25-30 分鐘 |


## 🎵 音樂功能使用

1. 點擊右下角 🎵 按鈕打開音樂面板
2. 選擇音樂來源：
   - **上傳本地音樂**：直接上傳 MP3/WAV 文件

   - **尋找免費背景音樂**：進入指南，直接連結到 3 個免費音樂網站
3. 調整音量並享受背景音樂
