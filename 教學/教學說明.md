# 📚 RPG 遊戲製作教學資源

> 歡迎來到「AI 驅動的 RPG 遊戲製作」完整教學課程！<br>
> 本資料夾包含從零到有製作一個完整 RPG 遊戲所需的所有教材。

---

## 🎯 課程概況

### 課程目標
- 學習如何用 AI 工具（Gemini、NotebookLM）快速生成遊戲內容
- 理解遊戲設計和架構的基本原理
- 實現一個功能完整的 RPG 教育遊戲
- 從企劃到發布的完整流程

### 適合人群
- 🎓 高中/大學學生
- 👨‍🏫 想用遊戲教學的老師
- 💻 對遊戲開發感興趣的人
- 🤖 想學習 AI 應用的創意工作者

### 時間投入
- **快速版**（只體驗工具）：2-3 小時
- **標準版**（完成一個遊戲）：30-40 小時（4 週）
- **進階版**（添加完整功能）：60+ 小時

### 技術需求
- **難度**：初中級（無需遊戲開發經驗）
- **工具**：都是免費的！
  - Gemini（Google 免費 AI）
  - NotebookLM（Google 免費工具）
  - React（免費開源）
  - GitHub（免費版控）

## 📖 5 份教學檔案介紹

### 1️⃣ **提示詞範本集合** 📋
**檔案**：`01_提示詞範本集合.md`

**內容**：
- 5 種核心提示詞範本（人物、故事、屬性、圖片等）
- 提示詞撰寫技巧
- 常見問題排查
- 版本追蹤表

**適合誰**：
- 想快速生成遊戲內容的人
- 對 AI 提示詞感興趣的人
- 需要參考範本的創作者

**使用方式**：
```
1. 複製對應的範本
2. 填入 {變數}
3. 貼到 Gemini/ChatGPT
4. 獲得生成結果
5. 按需調整和迭代

⏱️ 時間：1 小時完全掌握
```

**關鍵概念**：
- Level 1：人物設定生成
- Level 2：故事情節生成
- Level 3：屬性系統設計
- Level 4：圖片提示詞設計
- Level 5：提示詞微調技巧

## 💡 使用這些檔案的秘訣

### ✅ 推薦做法

1. **建立快捷方式**
   ```
   Bookmark 這個教學資料夾
   把常用檔案 pin 到編輯器
   建立快速查詢表格
   ```

2. **邊做邊學**
   ```
   不要只讀，邊讀邊實踐
   複製程式碼馬上執行
   試著改改數字看會發生什麼
   ```

3. **記錄心得**
   ```
   建立自己的筆記
   記錄遇到的問題和解決方案
   寫下你的改進想法
   ```

4. **分享和討論**
   ```
   和朋友/同學討論內容
   分享你做出來的結果
   一起協作製作遊戲
   ```

### ❌ 常見錯誤

- ❌ 只讀不做
  → 遊戲開發需要實踐，邊讀邊做效果最好

- ❌ 跳過基礎直接看代碼
  → 先理解設計概念，代碼才有意義

- ❌ 期望完美的第一版
  → 從 MVP 開始，逐步改進

- ❌ 不嘗試調整和實驗
  → 這些範本只是起點，鼓勵你創意改造
