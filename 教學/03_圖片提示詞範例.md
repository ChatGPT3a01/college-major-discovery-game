## 📋 第六部分：提示詞使用指南

### 6.1 提示詞最佳實踐

#### ✅ 做這些：

```
1. 具體而細節化
   ✓ "35-40 year old Taiwanese female engineer with black shoulder-length hair"
   ✗ "A woman"

2. 包含藝術風格
   ✓ "Professional digital illustration, artstation trending"
   ✗ 不指定風格

3. 指定品質和解析度
   ✓ "8K, high quality, professional, detailed"
   ✗ 遺漏品質指標

4. 使用顏色和光線
   ✓ "Warm golden hour lighting, blue and orange color scheme"
   ✗ 沒有光線描述

5. 提供背景與脈絡
   ✓ "Standing in modern engineering lab with monitors and blueprints"
   ✗ "在辦公室"
```

#### ❌ 避免這些：

```
1. 模糊的描述
   ✗ "A nice picture"

2. 過度複雜的句子
   ✗ 一個冗長的段落

3. 相互矛盾的指令
   ✗ "Photorealistic和cartoon style同時"

4. 過多特定的角色請求
   ✗ "看起來像某某明星"（著作權問題）

5. 技術上不可能的請求
   ✗ "同一個人的前後兩個時間點"
```

### 6.2 迭代改進流程

**第一次生成**：
```
使用基本提示詞
↓
評估結果（色彩、品質、風格、准確性）
↓
記錄哪些部分需要改進
```

**改進提示詞**：
```
基於評估結果修改提示詞
- 如果色彩不對：調整顏色描述
- 如果風格不符：改變藝術風格關鍵詞
- 如果細節不足：添加更多描述性語言
- 如果構圖不理想：加入位置和姿勢指令
```

**再次生成**：
```
使用改進後的提示詞
↓
比較結果與預期
↓
繼續迭代或完成
```

### 6.3 根據不同 AI 工具調整

#### Gemini 專用提示詞調整

```
更強調：
- 情感和故事性
- 多樣化與包容性
- 自然與溫暖的美感
- 教育背景

例：加入 "inclusive, warm, educational, diverse representation"
```

#### DALL-E 專用調整

```
強調：
- 清晰的構圖指令
- 明確的風格參考
- 技術性的視覺描述
- 照度和陰影

例：加入 "centered composition, professional photography, dramatic lighting"
```

#### Midjourney 專用調整

```
使用：
- --ar (寬高比) 參數
- --style raw (自然風格)
- 明確的相機視角
- 色溫指定

例：
"professional portrait --ar 3:4 --style raw --niji 6"
```

## 🔧 第七部分：快速參考表

### 常用術語翻譯

| 英文 | 中文 | 用途 |
|------|------|------|
| illustration | 插畫 | 藝術風格 |
| professional | 專業 | 品質層級 |
| digital painting | 數位繪畫 | 技法 |
| trending | 趨勢 | 品質提升 |
| artstation | 藝術站參考 | 品質參考 |
| warm lighting | 溫暖光線 | 氛圍 |
| 8K, 4K | 高清 | 解析度 |
| character art | 角色藝術 | 類型 |
| concept art | 概念藝術 | 類型 |
| vector | 向量 | 圖表/圖示 |

### 顏色方案參考

```
溫暖方案（鼓舞人心）
- 主色：金色、橙色、紅色
- 輔色：米色、棕色
- 用途：環保、創新、領導力

冷色方案（專業信任）
- 主色：藍色、靛色、紫色
- 輔色：白色、灰色
- 用途：技術、分析、醫療

活力方案（年輕活潑）
- 主色：綠色、青色、粉色
- 輔色：白色、黃色
- 用途：創意、好奇心、新鮮感

平衡方案（全能）
- 主色：紫色配金色
- 輔色：海軍藍、米白
- 用途：領導力、平衡屬性
```

## 🎯 實作建議流程

```
1️⃣ 先生成所有角色立繪
   ├─ 5-10 位導師
   └─ 預計時間：1-2 小時

2️⃣ 生成重要場景
   ├─ 開場場景
   ├─ 3-5 個關鍵故事場景
   └─ 預計時間：2-3 小時

3️⃣ 生成 UI 元素
   ├─ 屬性圖示
   ├─ 按鈕樣式
   └─ 預計時間：1 小時

4️⃣ 微調與迭代
   ├─ 根據測試反饋改進
   ├─ 保持風格一致性
   └─ 預計時間：2-3 小時

📊 總時間：6-9 小時即可生成完整視覺資源
```
