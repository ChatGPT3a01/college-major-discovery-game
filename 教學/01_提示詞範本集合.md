# RPG 遊戲製作：AI 提示詞範本集合

> 本文件包含從零到有製作 RPG 遊戲所需的提示詞範本。老師可直接複製使用。

---

## 🎭 第一階段：人物設定生成

### 範本 1.1：生成導師人物檔案

```
你是遊戲設計師。我正在製作一個 RPG 教育遊戲，幫助大學學生探索適合的科系。

請根據以下資訊，生成一個獨特的導師角色：

【輸入參數】
- 導師名字：{名字}
- 專業領域：{領域}
- 性格特徵：{特徵}
- 代表屬性：{屬性列表，如：領導力、創意力、同理心}

【要求輸出】
1. 人物簡介（2-3 句話）
2. 專業背景（3 點要點）
3. 教學風格（個性化教學方法）
4. 標誌性對白（1-2 句特色台詞）
5. 與玩家的互動方式（如何影響屬性）

【範例】
生成一個名叫「陳美琪」、專業領域是「環保工程」、性格特徵是「熱血實踐家」、代表屬性是「領導力、勇氣、同理心」的導師。
```

**使用場景**：快速生成新導師或豐富現有導師設定

## 📖 第二階段：故事分支生成

### 範本 2.1：生成故事情節

```
【遊戲背景】
這是一個大學科系探索遊戲，玩家跟隨不同導師學習，每次選擇會影響 6 項屬性：
- 創意力、分析力、領導力、同理心、勇氣、好奇心

【故事設定】
導師：{導師名字}
領域：{領域}
當前章節：{章節名稱}
故事情境：{情境描述}

【要求】
生成 3 個故事分支，每個分支包含：

1. **選項 A: {選項名稱}**
   - 故事結果（1-2 句）
   - 屬性變化：創意力 +{數字}，分析力 +{數字}...
   - 下一步提示

2. **選項 B: {選項名稱}**
   - 故事結果（1-2 句）
   - 屬性變化
   - 下一步提示

3. **選項 C: {選項名稱}**
   - 故事結果（1-2 句）
   - 屬性變化
   - 下一步提示

【語氣要求】
- 互動式、有趣、鼓勵探索
- 每個選擇都有邏輯後果
```

**使用場景**：快速生成故事樹，加速遊戲內容開發

### 範本 2.2：生成對話場景

```
角色對話生成請求

【場景設定】
- 導師：{導師名字}
- 場景：{具體情景}
- 玩家背景：{玩家角色描述}
- 對話目的：{目的，如「鼓勵、教導、挑戰」}

【對話要求】
請生成一段 8-12 行的對話，包含：

導師開場白（1-2 句，建立情境）：
> [導師說話]

玩家反應（玩家說話）：
> [玩家選項 A/B/C]

導師回應（根據玩家選擇）：
> [導師針對性回應]

屬性影響（最後標註）：
- 這段對話會增加：{屬性} +{數字}

【風格要求】
- 自然、流暢
- 包含導師的個性特色
- 有教育意義但不說教
```

## 🔄 第三階段：屬性系統設計

### 範本 3.1：屬性成長邏輯

```
我在設計一個遊戲屬性系統。請幫我驗證邏輯。

【屬性定義】
系統有 6 項屬性：

| 屬性名稱 | 定義 | 相關職業 |
|---------|------|---------|
| 創意力 | 想像、創新、藝術表現 | {職業} |
| 分析力 | 邏輯、問題解決、數據 | {職業} |
| 領導力 | 決策、團隊協作、影響力 | {職業} |
| 同理心 | 溝通、理解他人、共感 | {職業} |
| 勇氣 | 冒險、挑戰、毅力 | {職業} |
| 好奇心 | 探索、學習、創新思維 | {職業} |

【請求】
1. 驗證這些屬性是否涵蓋主要職業需求
2. 建議每個屬性的「最佳職業搭配」（前 3 個）
3. 設計屬性組合的職業推薦（如創意+分析=UI設計師）
4. 提出平衡機制建議

【輸出格式】
以表格形式提供分析結果。
```

## 🎨 第四階段：圖片提示詞設計

### 範本 4.1：角色立繪生成提示

```
我需要為遊戲生成角色圖像。

【角色資訊】
- 名字：{名字}
- 職業：{職業}
- 年齡：{年齡}
- 性格：{性格}
- 代表色：{顏色}
- 風格：{藝術風格，如「溫暖插畫風」「Q版」}

【圖像提示詞生成要求】
請生成適合 AI 圖像生成工具（如 DALL-E、Midjourney）的詳細提示詞：

1. **角色整體描述**（包含：姿態、服裝、氣質）
2. **背景建議**（適合角色的場景）
3. **色彩方案**（配色建議）
4. **藝術風格**（參考風格）

【提示詞範例格式】
A professional portrait of {name}, {age}-year-old {profession}, {personality traits},
wearing {clothing details}, in a {background style} setting,
art style: {art style}, warm lighting, professional illustration, highly detailed...

【注意】
提示詞應該：
- 具體、詳細
- 避免歧義
- 包含風格參考
```

## 📋 第五階段：內容整合檢查清單

### 使用提示詞時的檢查點

```
在使用每個提示詞範本前，確認：

□ 所有 {變數} 都已填入具體數值
□ 清楚定義了任務的背景和目的
□ 指定了輸出格式和長度要求
□ 設定了語氣和風格偏好
□ 包含了品質標準或限制條件

【常見問題排查】

Q: AI 回應太長/太短？
A: 在提示詞中明確指定「生成 X 句話」或「1-2 段落」

Q: 內容與遊戲設定不符？
A: 在提示詞開頭詳細說明遊戲背景、規則、屬性系統

Q: 屬性數值不合理？
A: 提供屬性範圍參考（如 +5 ~ +20）和平衡目標

Q: 角色風格不一致？
A: 加入「保持與其他角色的一致性」、「參考已有角色風格」的要求
```

## 🚀 提示詞微調技巧

### 5 步驟改進流程

```
1. 【執行】用範本生成初版內容
   ↓
2. 【評估】檢查品質、邏輯、風格一致性
   ↓
3. 【調整】根據評估結果修改提示詞
   - 加入更多約束條件
   - 提供反面例子（什麼不要）
   - 添加參考風格
   ↓
4. 【重新生成】用改進後的提示詞重新執行
   ↓
5. 【迭代】重複直到滿意

【改進提示詞的方法】

原版：
> 請生成一個教師角色

改進版：
> 請生成一個 35-40 歲的工程系教授角色。
> 性格應該：務實、有幽默感、鼓勵學生冒險
> 風格應該與「陳美琪」（環保工程師）一致，但領域不同
> 避免：刻板印象、說教、過度正式
> 包含：成就亮點、個人故事、給學生的建議
```
